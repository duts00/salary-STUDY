### `salario.py`
Calcula o novo salário de um funcionário com base no cargo e percentual de aumento:
- Cargos:
  - GER: 10%
  - ENG: 20%
  - TEC: 30%
- Se o cargo não estiver na tabela, aplica 40% de aumento.
- Mostra salário antigo, aumento, salário final e diferença.

**Uso:**
```bash
python3 salario.py
